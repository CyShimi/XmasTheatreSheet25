# XmasTheatreSheet25

Interactive “Theatre Sheet” web app for the Xmas 2025 project:
- Click nodes to open a song panel
- Artwork background per song
- Lyrics per song (clean text, stream-friendly)

## Live
Enable GitHub Pages in Settings → Pages, then your link will appear here.

## Local run
Open `index.html` in a browser (recommended: Chrome/Firefox).

## Notes
This repo is static (no server). Assets live in `assets/`.
